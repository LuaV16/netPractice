# NetPractice

## Description

The objective of this project is to introduce students to the fundamentals of computer networking. The goal of the project is to solve a series of 10 practical networking problems using a web-based training interface provided by 42.

Through these levels, the project covers the configuration of IP addresses, subnet masks, and routing tables to ensure effective communication between different nodes (clients, servers, switches, and routers).

---

## Instructions

The project does not require any installation, only the downloaded project files and a standard web browser are needed. To pass the project, 10 exported configuration files (one per level) must be placed at the repository root.

## Resources

### Networking Concepts Studied
In order to solve the configuration puzzles in this project, the following networking concepts were studied and applied:
* **TCP/IP Addressing:** Mastering IPv4 structure, understanding how network and host portions are divided, and identifying public vs. private IP ranges.
* **Subnet Masks & CIDR Notation:** Calculating subnets, determining the number of valid host IPs available in a block, and identifying network and broadcast addresses.
* **Default Gateways:** Configuring the correct next-hop IP address that a host or node must use to forward traffic outside its local network.
* **Routers and Switches:**
  * *Switches:* Operating at Layer 2 to facilitate communication between devices within the same local area network (LAN) using MAC addresses.
  * *Routers:* Operating at Layer 3 to route data packets across different subnets based on IP routing tables.
